# 🚀 Freelancer Dashboard

A comprehensive web-based dashboard built with **Flask** to help freelancers manage their projects, track earnings, and gain insights through interactive data visualizations.

![Python](https://img.shields.io/badge/Python-3.11-blue?style=flat&logo=python)
![Flask](https://img.shields.io/badge/Flask-3.1.1-black?style=flat&logo=flask)
![License](https://img.shields.io/badge/License-MIT-green.svg)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔐 **User Authentication** | Secure signup & login system with role selection |
| 📋 **Project Management** | Add, edit, delete & track freelance projects |
| 💰 **Revenue Tracking** | Monitor earnings across multiple platforms |
| 📊 **Data Visualization** | Interactive charts powered by Chart.js |
| 📱 **Responsive Design** | Clean dark-themed UI with sidebar navigation |

### Charts Included:
- 📊 **Revenue Bar Chart** — Earnings per project
- 📈 **Project Timeline** — Track deadlines over time
- 🥧 **Role Distribution Pie Chart** — 3D Artist, Web Developer, Content Writer
- 🎯 **Platform Radar Chart** — Fiverr, Upwork, Facebook distribution
- 🍩 **Status Polar Chart** — Completed, Pending, Failed

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** SQLite
- **Charts:** Chart.js
- **Styling:** Custom CSS with dark theme

## 🚀 Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/Salman-1210/freelancer-dashboard.git

# 2. Navigate to project directory
cd freelancer-dashboard

# 3. Create virtual environment (recommended)
python -m venv venv

# 4. Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 5. Install dependencies
pip install -r requirements.txt

# 6. Run the application
python app.py
